0    创建仓库            mkdir 仓库名称
0.1  进入目录            cd 进入目录名称
0.2  显示当前目录        pwd                   


1    初始化仓库          git init
1.1  添加文件到仓库      git add file名称                 可以反复使用
1.2  提交                git commit -m "摘要"

2    修改、添加、提交
2.1  修改                vi  file名称
2.2  查看仓库当前状态    git status 
2.3  查看difference      git diff file名称  显示格式正是Unix通用的diff格式
2.4  添加，提交同上

3    版本回退
3.1  显示提交日志        git log --pretty=oneline
